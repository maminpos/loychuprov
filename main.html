Приведенный код реализует простую линейную регрессию для построения модели связи между двумя переменными 'x' (годы) и 'y' (инвестиции).
Для вычисления коэффициентов линейной регрессии используются следующие:
1. Среднее значение x_mean вычисляется путем суммирования всех значений переменной x и деления на их количество.
2. Среднее значение y_mean вычисляется путем суммирования всех значений переменной `y` и деления на их количество.
3. Числитель numerator вычисляется путем суммирования произведений разностей (curr - x_mean) * (y[i] - y_mean) для каждого значения x и соответствующего значения y.
4. Знаменатель `denominator` вычисляется путем суммирования квадратов разностей `(curr - x_mean)` для каждого значения x.
5. Коэффициент наклона (`slope`) вычисляется делением numerator на denominator.
6. Коэффициент смещения (`y_intercept`) вычисляется путем вычитания произведения slope * x_mean из y_mean.
7. Массив `regression_line` строится путем вычисления значения линейной регрессии для каждого значения `x` с помощью формулы `slope * year + y_intercept`.
В итоге, в коде создается график, на котором отображаются исходные данные и линия линейной регрессии. Это позволяет визуализировать и оценить связь между переменными и построить прогнозные значения на основе модели линейной регрессии.
